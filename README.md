# 🛒 SQT Mart - Case Study (E-commerce Platform)

## 📌 Project Overview
SQT Mart is a real-world e-commerce grocery platform that allows users to browse and order frozen and dry food products online. It provides a smooth shopping experience along with a powerful admin system for managing products, orders, and users.

This repository is a **case study documentation** of a production-level system I worked on.

---

## 🧩 Key Features
- Product browsing with categories (Frozen & Dry foods)
- Shopping cart & order management system
- User authentication & account management
- Admin dashboard for product & order management
- Location-based delivery system
- Responsive UI for all devices
- Inventory & stock handling

---

## 🛠️ Tech Stack
- React.js (Frontend)
- Laravel (Backend)
- MySQL (Database)
- REST APIs
